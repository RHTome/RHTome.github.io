学习逻辑

大脑会毫不客气地丢掉那些它认为没用的东西。如果一个东西你几乎不使用它，那么它就是没用的。对于2天前学习的内容，在没有复习的情况下，还能记住1/4就已经很不错了。如果你持续地复习，那情况就不一样了。

小型卡片

让卡片尽量简单，专注于元数据，不可约数据，无组织数据。大卡片中存在多个目标，会分散注意力。大脑往往只能记住其中的一部分，并且忘得很快。在一次次的重复中，大卡片会大肆地消耗我们的时间和精力，然而像这样的犯罪很多人并不能意识到。

这很像扫射和点射的区别。在扫射模型中，只要还有一个敌人，就进行全场扫射。在点射模型中，采取的是各个击破的方式。

If you proceed with learning using spaced repetition, the longer the time you need to remember knowledge, the more you benefit from simplifying your items!

小型答案

寻找简单的提问方式，使回答成为元数据。

We want answer to be as short as imaginably possible!

并非只有问号用于提问，空格也是很好的选择。它可以提供语境，同时保证答案的简洁性。空格在增量阅读思想中是作为核心概念而存在的。

Cloze deletion makes the core of the fast reading and learning technique called incremental reading. The key to creativity is an association of remote ideas. By studying multiple subjects in unpredictable order, you will increase your power to associate ideas. This will immensely improve your creativity.

If you hold down alt (option on a Mac) while creating a cloze, Anki will automatically use the same number instead of incrementing it. Cloze deletions don’t need to fall on word boundaries, so if you select “anberra” rather than “Canberra” in the above example, the question would appear as “C[…] was founded in 1913”, giving you a hint.

If you can’t answer within about 10 seconds, it’s probably better to give up and show the answer than keep struggling to remember.

基于语义理解

Don’t memorize without understanding: If you’re studying a language, try to avoid large lists of words. The best way to learn languages is in context, which means seeing those words used in a sentence. Likewise, imagine you’re studying a computer course. If you attempt to memorize the mountain of acronyms, you’ll find it very difficult to make progress. But if you take the time to understand the concepts behind the acronyms, learning the acronyms will become a lot easier.

Many people say:”Anki helps you remember everything.” No, Anki is a tool, and every tool has manual. Anki is useful for remenbering what anki likes.

最小信息原则

语境压缩不仅包括对内容在空间上的压缩（裁除冗余、信息总结、缩写助记、基于关键词引导的语境），还包括对内容在时间上的压缩。当文字语境转换为图像语境时，阅览信息的时间成本将下降。

卡片生成逻辑

Notes 和 Fields

Anki 中的数据主要分为 Notes（记录） 和 Fields（字段）。一条记录包含了每种字段的一个取值。

Field对于大小写敏感。

Fields are stored in HTML. When you include formatting in your fields, extra HTML is included. If you prefix a field name with text:, Anki will not include any formatting.

By keeping content in separate fields rather than stuck together in a field, you make it much easier to adjust the layout of your cards in the future.

使用文本导入时，文本里的一行就是一条记录，每条记录里面的各个字段用分隔符分隔。分隔符可以是逗号、分号或 tab ，Anki 会以第一条记录判断用什么分隔符，并判断有多少个字段。而字段的名称最后在要导入的那个牌组上面设置。

比如记忆英文，可以有三个字段：英文、中文、例句，文本中的一行记录就可以是：

Apple; 苹果; I love apple.

这里就代表了：

英文: Apple
中文: 苹果
例句：I love apple.

得到数据后，就可以任意地生成卡片。设置你自己希望在正面看到的内容以及希望在背面看到的内容。

We add notes rather than cards. Anki creates cards for us.

Notes and note types are common to your whole collection rather than limited to an individual deck. This means you can use many different types of notes in a particular deck, or have different cards generated from a particular note in different decks.

When Anki checks for duplicates, it only compares other notes of the same type. Thus if you add a capital city called “Orange” using the capital city note type, you won’t see a duplicate message when it comes time to learn how to say “orange” in French.

When Remember last input… is checked, Anki will not clear out this field’s content after a note is added. If you find yourself entering the same content into multiple notes, you may find this useful.

These related cards are called siblings. When you answer a card that has siblings, Anki can prevent the card’s siblings from being shown in the same session by automatically burying them. Buried cards are hidden from review until the clock rolls over to a new day or you manually unbury them using the “Unbury” button that’s visible at the bottom of the deck overview screen. Anki will bury siblings even if the siblings are not in the same deck (for instance, if you use the deck override feature). Anki will only bury siblings that are new or review cards.

使用手册

Decks

牌组可以构成树形结构，形成父子关系。在牌组“英语::单词”中，”单词“就是“英语”的子牌组。子牌组属于父牌组，在学习父牌组时，子牌组中的卡片也会显示，否则树形结构无意义。

如果“英语”和”单词“是两个平行牌组，你可以将”单词“拖动到“英语”牌组之下使其成为“英语”的子牌组，或者通过设置将”单词“牌组重命名为英语::单词。

你最好自己创建牌组，使创建的笔记记录问题的语境。这回促使你认真做笔记，思考真正重要的点，并获得自己的理解。如果你去下载单词书来背，并非是一条好的道路，你应该接触大量的实际表达，从中学习。

Do not learn if you do not understand. —SuperMemo

在记忆牌组时，按s键即可回到牌组列表。

将牌组分类太细会导致复习牌组时相关性强的笔记相继出现，你甚至可以通过上下文猜出答案，这无疑会干扰记忆，削弱记忆效果。一般情况下请不要将相关性强的内容一起记忆，你可以使用tag或field来进行笔记分类，而不要创建太多的牌组。

工具菜单中学习牌组可以快速搜索牌组，用/触发，使用空格以与逻辑联结多个搜索词。

Review

anki会严格遵守你的设置。

当设置了一天最大复习卡片数量，那么一旦超量，即使卡片已经到了该复习的时候，anki也不予显示。

当设置了一天最大学习卡片数量，卡片会累积，但不会超量。

若父牌组一天最大学习卡片数量为A，那么本身及所有子牌组的学习总和不会超过A，因为所有子牌组在内容上属于父牌组。

你越急，越不能长久，一天之内饱学，必然无法承受翻数倍之复习量。

如果今天要学的其他卡片学完了，那些需要今天复习但还未到复习时间的卡片怎么办？Anki默认直接进行显示。不过可以在设置中更改，比如你想等时间到了再学。会当天还没到复习时间的卡片也会被显示。

Anki默认显示卡片下次复习的时间，也可以设置为不显示。对于顺序添加的卡片，它们在被给与相同的记忆打分后拥有同步的延迟，从而仍然保持连续，而这会影响记忆效果。为应对这种情况，anki会在回答打分后，默默地为延迟增加一定范围内的随机值。

如果你没落后于复习进度，anki会优先显示那些被拖延得最久的卡片，这是为了防止有些卡片会被无限制地拖延下去。除非你将过去的债都还清，否则anki是不会对新学卡片进行复习的。
